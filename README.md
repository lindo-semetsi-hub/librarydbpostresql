# librarydbpostresql

This project5 is a basic Library Management System built using PostgreSQL. It manages books, authors and patrons. The system supports adding records, updating them, deleting and running different queries.

1. CREATE TABLES:
   - CREATE TABLE authors (...)
   - CREATE TABLE books (...)

2. INSERT DATA:
