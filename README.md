## librarydbpostresql

# This project is a basic Library Management System built using PostgreSQL, divided into 5 sprints. It manages books, authors and patrons. The system supports adding records, updating them, deleting and running different queries. 

## COMMANDS
1. CREATE TABLE
2. INSERT INTO
3. SELECT
4. UPDATE
5. DELETE FROM

  
# How to run queries in pgAdmin
1. Open pgAdmin
2. Connect to the PostgreSQL server
3. Right click the database, opening Query Tool
4. Paste the SQl command
5. Click run

# How to run queries in psql
1. Open the terminal and connect:
   - psql -U username -d LibraryDB
2. Run a SQL file:
   - \i filename.sql
Alternatively, you can just paste SQL directly into the terminal.
